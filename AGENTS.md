# Layout

- **`~/.cursor/rules/`** — Project rules (`.mdc` files) for Cursor’s rules system.
- **`~/.cursor/commands/`** — Slash commands (e.g. `/plan`, `/tdd`, `/learn`).
- **`~/.cursor/agents/`** — Agent prompt definitions used by commands and workflows.
- **`~/.cursor/skills/`** — Skills; each skill is a folder containing `SKILL.md` (and optional `reference/`, scripts, etc.).
